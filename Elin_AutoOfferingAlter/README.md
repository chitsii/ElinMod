# Elin_AutoOfferingAlter

睡眠時に中身のアイテムを信仰している神へ奉納するコンテナを追加します。

## 機能
- **信仰の箱 (Offering Box)**:
  - 「簡易制作」で作成可能
  - インベントリに「信仰の箱」を所持して睡眠をとると、箱内の全ての奉納可能なアイテムを信仰中の神に奉納します。
  - 信仰ポイントを最大化するため、一度に奉納するスタック個数を最適化します
    - 1回あたりの奉納効果量が1500ポイント以上になる最小の個数を計算し、それを基準に分割
    - スタック数が基準の個数で割り切れない場合、まず余り個数を先に奉納します
        - 例：効果量10の肉が330個ある場合（目標150個=1500ポイント）→ まず30個（端数）を奉納し、その後に150個を2回奉納

## アンインストール時の注意
事前に箱の中身をすべて取り出さないと中身が消えてしまうのでご注意ください。
（MODを削除すると、セーブデータ内に存在する「信仰の箱」が「錬金灰」アイテムに変化します）
