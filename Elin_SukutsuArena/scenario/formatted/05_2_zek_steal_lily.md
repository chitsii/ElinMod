# クエスト：ゼクの囁き『偽りの器、真実の対価』

**【概要】**
リリィに納品する器を手に持つプレイヤーの前に、ゼクが現れる。彼は「本物の器を自分に渡し、偽物をリリィに渡せ」と囁く。これは重大な裏切りであり、プレイヤーの道徳心と野心が試される分岐点となる。

**【登場人物】**
- **ゼク**：次元の狭間を漂う商人。プレイヤーに裏切りを提案する。
- **リリィ**：受付のサキュバス。器の完成を待っているが、ゼクの陰謀には気づいていない。

---

## 【シーン1：影からの招き】

### 演出・背景描写

**[BGM: 突然の静寂 → 不穏な弦楽器の音]**

リリィが納品された器の最終チェックのため、一時的に席を外した隙だった。

*【視覚演出：ロビーの灯火が不自然に揺らぎ、影が伸びる】*

ロビーの灯火が不自然に揺らぎ、あなたの足元の影が、まるで意思を持った沼のように長く伸びる。

*【聴覚演出：鎖の擦れる音、金属的な響き】*

そこから、鎖の擦れる音と共に、ゼクが音もなく這い出してきた。

### ダイアログ

**ゼク**：
「……素晴らしい。実に、溜息が出るほどに。」

*【表情：フードの奥から、愉悦に満ちた視線】*

**ゼク**：
「あのサキュバスに渡すには、あまりに惜しい『傑作』を作られましたね、闘士殿。」

彼は細長い指で、あなたが持つ器を指し示す。

**ゼク**：
「彼女が何と言ったかは知りませんが……あの女がその器で集めようとしているのは、単なる『音』ではありません。」

*【表情：冷笑、フードの奥で何かを愉しむような笑み】*

**ゼク**：
「そこに溜まるのは、あなたの戦いが生み出した『運命の雫』……。
彼女はそれを啜り、あなたの魂をより効率的に『管理』しようとしているのですよ。
……ふふ、お気づきでしたか？」

### プレイヤーの選択肢（PR用）

**選択肢A**：「……何が言いたい」
→ ゼク：「おや、警戒心がおありで。では、単刀直入に申し上げましょう。」

**選択肢B**：「リリィが俺を管理しているだと？」
→ ゼク：「ええ、そうです。彼女はサキュバス。魂を喰らう存在です。あなたは気づいていなかったのですか？」

**選択肢C**：「（無言で聞く）」
→ ゼク：「……ふふ、沈黙は賢明さの証。では、続けさせていただきましょう。」

## 【シーン2：裏切りの提案】

### 演出・背景描写

**[BGM: 不協和音が強まる、心臓の鼓動のような低音]**

ゼクは長い指で、懐から一つの「偽物」を取り出した。

*【視覚演出：偽物の器が現れ、その内部に黒い霧が渦巻く】*

見た目はあなたが作った共鳴瓶と瓜二つだが、そのガラスの奥には、澱んだどす黒い霧が渦巻いている。

### ダイアログ

**ゼク**：
「どうでしょう、一つ提案があります。」

*【表情：フードの奥で、誘惑的な笑み】*

**ゼク**：
「その本物の器を、私に譲ってはいただけませんか？
代わりに、この**『呪われた模造品（ミミック・ボトル）』**を彼女に渡すのです。」

彼は偽物の器をあなたの前に差し出す。

**ゼク**：
「彼女は気づかないでしょう……。
いえ、気づいた時には、彼女の『研究』はあなたの支配ではなく、私への捧げ物へとすり替わっている。」

*【表情：愉悦に満ちた笑み、フードの奥で何かを企む目】*

**ゼク**：
「そしてあなたには、その誠実な裏切りの対価として、
私が異次元のゴミ捨て場で拾い上げた『本物の禁忌』を差し上げましょう。
……それと、特別な報酬も、ね。」

### プレイヤーの選択肢（PR用）

**選択肢A**：「その『禁忌』とは何だ？」
→ ゼク：「ふふ、興味を持たれましたか。では、お見せいたしましょう。」

**選択肢B**：「リリィを裏切れと？」
→ ゼク：「裏切りとは聞こえが悪い。……『戦略的な選択』と呼びましょう。」

**選択肢C**：「……代償は何だ」
→ ゼク：「おや、慎重ですね。代償は……あなたの『良心』を、少しばかりいただくだけです。」

## 【シーン3：天秤にかけられる魂（プレイヤーの選択）】

### 演出・背景描写

**[BGM: 緊張感のある音楽、時計の音]**

ゼクは「本物の器」と「偽物の器」、そして報酬となる「禍々しい品」を並べ、あなたの返答を待つ。

*【視覚演出：画面が二つに分かれ、選択肢が強調される】*

彼のフードの奥にある瞳が、あなたの決断を愉しむように細められた。

### ダイアログ

**ゼク**：
「リリィに従い、大人しく彼女の飼い犬となるか。
それとも、私と手を組み、彼女を欺いて『真なる力』を手にするか。」

*【表情：期待に満ちた笑み】*

**ゼク**：
「……ふふ、どちらを選んでも、このアリーナに刻まれるあなたの物語は、
より残酷で美しいものになるでしょう。
さあ、お選びください。」

---

### 【重要な選択】

**選択肢A：「断る。リリィを裏切るつもりはない」**

ゼクは肩をすくめ、偽物の器を影の中へ消す。

**ゼク**：
「……残念です。ですが、無理強いはいたしません。
あなたが『忠犬』の道を選ばれるというのなら、それもまた一興。
……いつか、その選択を後悔する日が来るかもしれませんが。」

*【表情：わずかに失望した表情、しかしすぐに笑みに戻る】*

ゼクは影の中へと消えていく。

**【効果】**：
- リリィとの信頼関係を維持
- 通常の報酬のみ

---

**選択肢B：「……分かった。取引に応じる」**

ゼクは満足げに本物の器を影の中へと消し、偽物をあなたの手に握らせる。

**ゼク**：
「ふふ、賢明な判断です。
これがあなたへの報酬……**小さなコイン10枚**と**プラチナコイン3枚**を、台帳に記録する手はずを整えておきましょう。
それと、この『影の印』を。」

*【視覚演出：ゼクの手から黒い光が放たれ、プレイヤーに触れる】*

**ゼク**：
「これで、私の店での取引が、より『有利』になりますよ。
……では、良い演技を。彼女に気づかれないよう、お気をつけて。」

*【表情：愉悦に満ちた笑み、フードの奥で何かを企む目】*

ゼクは影の中へと消えていく。

**【効果】**：
- カルマ -20（道徳的に問題のある行動）
- リリィの好感度：表面上は+10（後に発覚する可能性）
- フラグ設定：`chitsii.arena.player.bottle_choice = "swapped"`
- 報酬追加：小さなコイン × 10、プラチナコイン × 3
- ゼクの関係値：+15

---

## 【シーン4：事後の静寂】

### 演出・背景描写（選択肢Bを選んだ場合のみ）

**[BGM: 通常のロビーBGMに戻る]**

ゼクが消えた直後、リリィが戻ってきて、何食わぬ顔でその「偽物」を受け取る。

*【視覚演出：リリィが器を受け取り、満足げに微笑む】*

### ダイアログ

**リリィ**：
「お待たせしました。」

*【表情：柔らかい微笑み】*

彼女は器を手に取り、軽く傾ける。

**リリィ**：
「……あら、なんだか少し、器の感触が変わったかしら？」

*【表情：わずかに首を傾げる】*

一瞬、彼女の瞳があなたを鋭く見つめるが、すぐに笑顔に戻る。

**リリィ**：
「……まぁいいわ。これで私の研究は飛躍的に進みます。
ふふ、感謝してくださいね。あなたが『使い物』にならなくなった後も、その響きだけは私の手元に残るのですから。」

*【表情：意地悪な笑み、しかし疑念は消えていない】*

## 【システム処理】

**フラグ更新（選択肢Aの場合）**：
```json
{
  "chitsii.arena.player.bottle_choice": "refused",
  "chitsii.arena.rel.lily": "+10"
}
```

**フラグ更新（選択肢Bの場合）**：
```json
{
  "chitsii.arena.player.bottle_choice": "swapped",
  "chitsii.arena.player.karma": "-20",
  "chitsii.arena.rel.lily": "+10",
  "chitsii.arena.rel.zek": "+15"
}
```

**報酬（選択肢A：拒絶）**：
- 通常報酬のみ（05_1で既に受け取り済み）

**報酬（選択肢B：受諾）**：
- 小さなコイン × 10（追加）
- プラチナコイン × 3（追加）
- 「影の印」効果：ゼクの店での買い取り価格+20%

**重要な注意**：
この選択の結果は、**scenario/13_makuma2.md『シーン1.5：共鳴瓶の異変』**で回収されます。
選択肢Bを選んだ場合、リリィとの関係に重大な影響を及ぼす分岐イベントが発生します。

---

**【シナリオ意図】**

このシーンは、プレイヤーの道徳心と野心が試される重要な分岐点です。
- ゼクの誘惑的な提案と、リリィへの信頼のバランスを提示
- 短期的な利益（報酬）と長期的な関係性（信頼）のトレードオフ
- プレイヤーの選択が後のシナリオに大きな影響を与えることを暗示
- ダークファンタジーらしい「道徳的グレーゾーン」の選択を提供
