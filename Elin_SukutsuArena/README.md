# 地下闘技場 (Sukutsu Arena)

異次元の闘技場「ヴォイド・コロシアム」を舞台にした物語型クエストMod。

## 概要

神々の暇つぶしのために殺し合う闘士たちの世界。あなたは偶然この地に迷い込み、闘技場の頂点を目指すことになる。

### キャラクター

- **リリィ** - 魅惑の受付嬢（サキュバス）
- **バルガス** - 百戦の覇者（アリーナマスター）
- **グランドマスター** - 竜鱗の王者（最終ボス）
- **怪しい商人** - 闇市の支配者

## 動作要件

- Elin EA 23.x 以降
- [Custom Whatever Loader (CWL)](https://steamcommunity.com/sharedfiles/filedetails/?id=XXXXXX)

## インストール

1. Steam Workshop からサブスクライブ、または
2. `Package` フォルダを `Elin/Package/` にコピー

## アンインストール時の注意

> [!CAUTION]
> **セーブデータ保護のため、以下の手順を守ってください**

### 手順

1. **闘技場を出る** - 通常のゾーン（ダルフィ、野原など）に移動
2. **パーティを確認** - 闘技場NPCがパーティにいれば解散
3. **セーブして終了** - ゲームを保存してから終了
4. **Modを削除** - Workshop からアンサブスクライブ、またはフォルダを削除

### なぜ必要？

- 闘技場内にいる状態でModを削除すると、ゾーン読み込みエラーが発生する可能性があります
- フラグデータ（`dialogFlags`）はセーブに残りますが、無害です
