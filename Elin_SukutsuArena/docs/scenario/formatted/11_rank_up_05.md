# クエスト：Rank B 昇格試験『虚無の処刑人：心なき白刃』

**【概要】**
ランクB『銀翼（Silver Wing）』への昇格試験。プレイヤーは、バルガスが最も忌み嫌う「心なき刃」——異次元のゴミ捨て場で拾い上げられ、殺すためだけに調整された暗殺用自動人形「ヌル」——と対峙する。これは単なる戦闘ではなく、魂と虚無の対決である。

**【登場人物】**
- **バルガス**：アリーナマスター。ヌルを最も忌み嫌い、畏怖している。
- **リリィ**：受付のサキュバス。ヌルの恐ろしさを警告する。
- **ヌル：沈黙の執行者**：心なき暗殺用自動人形。完全隠密の権化。

## 【シーン3：闘技場：無音の深淵】

### 演出・背景描写

**[BGM: 静寂 → 心臓の鼓動音のみ]**

ゲートの先には、何もなかった。

*【視覚演出：一面、白く塗りつぶされたような空間、音がない】*

ただ一面、白く塗りつぶされたような、音のない空間。

*【聴覚演出：完全な静寂、自分の心臓の音だけが聞こえる】*

そこで待っていたのは、ボロを纏った細身の影。

*【視覚演出：白い仮面をつけた細身の人型、光を反射しない漆黒の短剣】*

顔には感情を読み取らせない白い仮面。
その手には、光を反射することすら拒む、漆黒の短剣が握られている。

*【聴覚演出：ヌルの呼吸音すら聞こえない、完全な静寂】*

### システム演出

**一騎打ち：【ヌル：沈黙の執行者】**

**エネミー特性**：
- **『不可知の影歩き』**：戦闘開始直後に姿を消す。こちらの「知覚」が低いと、位置を特定できない。
- **『絶息の刃』**：物理防御（PV）を完全に無視する。連続で被弾すると、即死カウントダウンが始まる。
- **『魔力拒絶』**：魔法の詠唱を感知すると、瞬時に背後に転移して「沈黙」を付与するカウンター。

**特殊ギミック：【心音の鼓動】**
敵が近づくほど、BGMから音が消え、プレイヤーの心音（ドクン、ドクンという鼓動音）だけが大きく響く。この音を頼りに、敵の距離を測る必要がある。

**戦闘中の演出**：
- ヌルが姿を消すと、画面が白く霞む
- ヌルが近づくと、心音が激しくなる
- ヌルの攻撃直前、一瞬だけ空間が歪む
- ヌルを攻撃すると、無表情のまま後退する

**ヌルの声**（戦闘中、機械的な音声）：
「……ターゲット、確認。」
「……殺害、開始。」
「……排除、完了。」

**【シナリオ意図】**

このシーンは、「心なき刃」との対峙という恐怖と緊張を提供する重要なポイントです。
- ヌルという「完璧な殺人機械」を提示し、プレイヤーの恐怖を煽る
- バルガスとリリィの珍しい心配と助言で、キャラクターに深みを与える
- 完全隠密という新しい戦闘形式を導入し、緊張感を高める
- 心音の鼓動というギミックで、プレイヤーの没入感を高める
- 虚無と魂の対決というテーマを提示し、物語に深みを与える
- プレイヤーの成長を称賛し、感情的なカタルシスを提供
