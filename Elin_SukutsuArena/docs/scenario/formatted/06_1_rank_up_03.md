# クエスト：Rank E 昇格試験『鉄屑の慟哭』

**【概要】**
プレイヤーは次なる試練として、バルガスの元相棒カインの成れの果てと戦うことになる。これは単なる昇格試験ではなく、バルガスの過去と向き合い、彼の未練を断ち切るための戦いである。

**【登場人物】**
- **バルガス**：アリーナマスター。かつての相棒カインの魂を解放してほしいと願う。
- **カイン：虚無の残響**：アリーナの記憶が具現化した、錆びついた騎士。心も言葉も失っている。
- **リリィ**：受付のサキュバス。戦闘後、報酬を授与する。

---

## 【シーン1：酒の切れた夜に】

### 演出・背景描写

**[BGM: 静寂、遠くで風が吹く音]**

ロビーの喧騒が嘘のように静まり返った深夜。

*【視覚演出：ロビーが薄暗く、松明の炎だけが揺れている】*

バルガスはいつもの酒瓶を持たず、代わりに血錆にまみれた「古い戦士の兜」を無造作に眺めていた。

彼がこれほどまでに静かなのは、あなたがここへ来てから初めてのことだった。

*【聴覚演出：静寂の中、バルガスの深い呼吸だけが聞こえる】*

### ダイアログ

**バルガス**：
「……来たか。」

*【表情：沈痛な面持ち、目は遠くを見ている】*

彼は兜を撫でながら、低く掠れた声で話し始める。

**バルガス**：
「おい、新入り。お前は『鉄』がなぜ錆びるか知ってるか？
手入れを怠るからじゃねえ。……持ち主の心が折れた時、鉄も一緒に死ぬんだよ。」

*【表情：苦しげな表情、拳を握りしめる】*

**バルガス**：
「かつて俺と一緒に地獄を這いずり回った相棒がいた。名はカイン。
俺たちは『鉄の意志』を掲げてこのアリーナに挑んだが……あいつは、グランドマスターの影に触れ、魂をこの異次元の『錆』に食い尽くされた。」

彼は兜を床に置き、深く息を吐く。

**バルガス**：
「あいつは……強かった。だが、守るべきものを失って、壊れちまった。
……俺は、あいつを救えなかった。」

*【表情：後悔と悲しみが混じった表情】*

### プレイヤーの選択肢（PR用）

**選択肢A**：「……辛い話だな」
→ バルガス：「辛い？ ……ああ、そうだな。だが、それがこのアリーナの現実だ。」

**選択肢B**：「カインは今どうなっている？」
→ バルガス：「……あいつは、この場所の『記憶』になっちまった。お前の次の相手だ。」

**選択肢C**：「（無言で聞く）」
→ バルガス：「……まあ、黙って聞いててくれ。続きがある。」

## 【シーン3：闘技場：追憶の墓場（戦闘開始）】

### 演出・背景描写

**[BGM: 戦闘BGM - 哀しげなメロディ、重い鼓動音]**

闘技場の門を潜ると、そこはいつもの砂地ではなく、無数の折れた剣と壊れた鎧が積み上がった「武具の墓場」へと変貌していた。

*【視覚演出：画面全体に赤錆の霧がかかり、視界が霞む】*

中央に立つのは、全身から赤黒い錆を滴らせ、顔のない兜から青い炎を揺らめかせる一人の騎士。

*【聴覚演出：金属が軋む音、低い唸り声】*

騎士は巨大な錆びた剣を地に突き立て、ゆっくりと頭を上げる。

*【視覚演出：カインの兜から青い炎が噴き出し、周囲の武具が震える】*

**【観客の声】**（遠くから）
「……ほう。あの『錆びた英雄』がまた召喚されたか」
「賭けは『3ターン以内に装備が崩壊』で。」
「いや、この新人は少しは保つだろう。『5ターン』だ。」

### システム演出

**マップギミック：【赤錆の霧】**
全装備の性能が一時的に20%低下する。さらに、物理攻撃を受けるたびに確率で「装備の劣化」が発生する。

**ターゲット**：
- 『カイン：虚無の残響』

**特殊行動**：
- **アイアン・レクイエム**：自身の周囲に折れた剣を召喚し、一斉に降り注がせる広範囲物理攻撃。

**戦闘中の演出**：
- HP50%以下で、カインが苦しげに唸る
- バルガスの声が闘技場内に響き渡る：「……カイン！ まだ眠ってんじゃねえ！ その剣を、その誇りを思い出せ！」
- カイン撃破時、青い炎が消え、粒子となって崩れ去る

## 【システム処理】

**フラグ更新**：
```json
{
  "chitsii.arena.player.rank": "E",
  "chitsii.arena.rel.balgas": "+15"
}
```

**報酬**：
- 小さなコイン × 8
- プラチナコイン × 3
- 選択した素材 × 1（鋼鉄の欠片 / 魔力の結晶 / 骨）

**ステータス変化**：
- 称号獲得：【鉄の意志を継ぐ者】（物理防御+10、装備劣化確率-50%）

**バルガスの変化**：
このクエスト以降、バルガスが「安酒」ではなく「少し高級な酒」を飲むようになり、プレイヤーへの態度に敬意が混じるようになる。

---

**【シナリオ意図】**

このシーンは、バルガスというキャラクターの深みを見せる重要なポイントです。
- 過去の友情と喪失、そして救済という感情的なテーマを提示
- プレイヤーが単なる「闘士」ではなく、「人々の未練を解放する者」としての役割を担う
- カイン戦の哀しい雰囲気と、バルガスの涙で、感情的なカタルシスを提供
- プレイヤーの選択肢を豊富に用意し、感情移入を促進
