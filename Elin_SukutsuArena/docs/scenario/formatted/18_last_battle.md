# 最終章：『虚空の王、静寂の断罪』

**【概要】**
プレイヤーはバルガス、リリィ、ゼクと共にアスタロトとの最終決戦に臨む。仲間たちの助けを借りて、プレイヤーは「レベル1億」の王を倒し、アリーナという牢獄を破壊する。そして、地上への道が開かれる。プレイヤーの選択により、複数のエンディングが用意されている。

**【登場人物】**
- **プレイヤー**：黄金の戦鬼、屠竜者。アスタロトに挑む。
- **バルガス**：アリーナマスター。最後の戦いに参加する。
- **リリィ**：受付のサキュバス。プレイヤーの魂を守る。
- **ゼク**：次元の狭間を漂う商人。プレイヤーの「奇跡」を見届けたいと願う。
- **アスタロト**：グランドマスター。レベル1億の王。
