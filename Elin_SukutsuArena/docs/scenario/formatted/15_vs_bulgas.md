# クエスト：Rank S 昇格試験『理を拒む者：師匠との最終決戦』

**【概要】**
ランクS『屠竜者（Dragon Slayer）』への昇格試験。バルガスが禁忌の薬を使い、全盛期の姿を取り戻してプレイヤーに挑む。しかし、真の試練は彼を倒すことではなく、アリーナの「殺せ」という命令を拒絶し、師匠を生かすことである。プレイヤーの人間性が最も試される重要なシーン。

**【登場人物】**
- **バルガス**：アリーナマスター。禁忌の薬で全盛期の姿を取り戻し、プレイヤーに挑む。
- **リリィ**：受付のサキュバス。バルガスの命を心配し、プレイヤーに懇願する。
- **ゼク**：次元の狭間を漂う商人。プレイヤーの選択を影から見守る。

## 【システム処理】

**フラグ更新**：
```json
{
  "chitsii.arena.player.rank": "S",
  "chitsii.arena.rel.balgas": "+50",
  "chitsii.arena.rel.lily": "+30",
  "chitsii.arena.player.system_breaker": "true"
}
```

**報酬**：
- 小さなコイン × 50
- プラチナコイン × 30
- 選択した素材 × 1（バルガスの剣の欠片 / エーテルの欠片 / 魔力の結晶）

**ステータス変化**：
- 称号獲得：【理を拒む者（System Breaker）】（状態異常耐性+30%、ピンチ時にバルガスの幻影が加勢）

**バフ付与**：
- **バルガスの加護**（永続・台帳記録）
  - 全スキルの消費コスト-20%
  - ピンチ時（HP30%以下）にバルガスの幻影が一時的に加勢

**アンロック**：
バルガスが戦闘から引退し、ロビーで常にプレイヤーを励ます「精神的支柱」となる。

---

**【シナリオ意図】**

このシーンは、プレイヤーがアリーナの「理」を拒絶し、真の人間性を示す最も重要なポイントです。
- バルガスとの戦いを通じて、プレイヤーの成長と師弟関係を描く
- アリーナの「殺せ」という命令を拒絶することで、プレイヤーが「システムのバグ」として完成する
- プレイヤーの初期動機（強欲/戦闘狂/虚無/傲慢）が、ここで「慈悲」へと昇華される
- リリィとバルガスへの愛情が、プレイヤーの選択を支える
- 最終決戦への布石として、プレイヤーの「人間性」が最大の武器であることを示す
