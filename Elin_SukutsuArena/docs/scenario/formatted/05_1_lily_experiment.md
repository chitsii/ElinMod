# クエスト：リリィの私的依頼『残響の器』

**【概要】**
リリィはアリーナに満ちる「死の残響」を収集するため、特殊な器の製作をプレイヤーに依頼する。これは単なる雑用ではなく、プレイヤーの「繊細な技術」を試す試練でもある。

**【登場人物】**
- **リリィ**：受付のサキュバス。研究者としての一面を見せ、プレイヤーに製作依頼を出す。
- **プレイヤー**：石細工またはガラス工芸のスキルを持つ闘士。

---

## 【シーン1：受付嬢の不機嫌な午後】

### 演出・背景描写

**[BGM: 不協和音、紙をめくる音、ペンが机を叩く音]**

ロビーは相変わらず、異次元の歪みが軋む不快な音に満ちている。

*【視覚演出：リリィの周囲に、紙の束が散乱している】*

リリィは眉間に皺を寄せ、羽根ペンを乱暴に机に置いた。
その前には、どこか禍々しい幾何学模様が描かれた、古ぼけた設計図が広げられている。

*【聴覚演出：遠くで戦いの音、観客の笑い声、次元の歪む音】*

### ダイアログ

**リリィ**：
「……あぁ、忌々しい。」

*【表情：苛立ち、尻尾が不機嫌そうに床を叩く】*

彼女は設計図を睨みつけ、ため息をつく。

**リリィ**：
「このロビーの『雑音』のせいで、私の大切な研究がちっとも進みません。
戦士たちの絶望、観客の哄笑、次元の摩擦音……これらは本来、純粋な『魔力の残滓』として回収されるべきものなのに。」

彼女はあなたを見つけ、わずかに目を細める。

**リリィ**：
「ねえ、そこの『泥犬』さん。
少しは私に牙以外の役に立つところを見せてくださる？」

*【表情：冷笑、しかし瞳には期待の色】*

**リリィ**：
「私の研究のために、特別な『実験用具』が必要なのです。
……そう、地上の安っぽい職人では形にすることすら叶わない、特別な器が。」

### プレイヤーの選択肢（PR用）

**選択肢A**：「どんな器が必要なんだ？」
→ リリィ：「ふふ、興味を持っていただけましたか。では、詳しくご説明いたしましょう。」

**選択肢B**：「報酬次第だな」
→ リリィ：「まあ、現実的ですこと。ええ、もちろん報酬はお支払いいたしますよ。」

**選択肢C**：「俺に製作技術があると思うのか？」
→ リリィ：「……どうでしょうね。試してみなければ分かりませんが、少なくとも期待はしていますよ。」

## 【シーン2：技術への要求（製作スキルの提示）】

### 演出・背景描写

**[BGM: 神秘的な音楽、鐘の音]**

リリィは細長い指先で設計図の一点を鋭く指し示した。

*【視覚演出：設計図がクローズアップされ、複雑な構造が見える】*

そこには、内部に複雑な空洞を持つ、特殊な瓶のような構造が描かれている。

### ダイアログ

**リリィ**：
「必要なのは**『虚空の共鳴瓶』**。
ただのガラスや石では、次元の圧力に耐えきれず粉々に砕けてしまうわ。」

*【表情：真剣な眼差し、尻尾が静止する】*

**リリィ**：
「あなたがもし、『石細工』あるいは『ガラス工芸』の心得があるのなら、
**黒曜石**か**高品質な水晶**を素材にして、これを作りなさい。」

彼女は設計図を指でなぞり、細部を説明する。

**リリィ**：
「素材そのものの品質も重要ですが、何よりあなたの『集中力』が試されます。
……ふふ、もし不出来なものを持ち込んだら、その器の代わりにあなたの頭蓋骨を加工して使うことになりますが、よろしいかしら？」

*【表情：冷たい微笑み、しかし冗談ではない目】*

### プレイヤーの選択肢（PR用）

**選択肢A**：「任せてくれ。作ってみせる」
→ リリィ：「ふふ、自信がおありで結構。では、期待しておりますね。」

**選択肢B**：「……頭蓋骨は勘弁してくれ」
→ リリィ：「まあ、ご心配なく。冗談ですよ。……多分。」

**選択肢C**：「具体的な要求仕様は？」
→ リリィ：「おや、職人気質ですこと。では、細部までご説明いたしましょう。」

## 【シーン3：製作の過程（プレイヤーへの指示）】

### 演出・背景描写

**[BGM: 作業BGM、静かで集中できる音楽]**

リリィは冷ややかな微笑を浮かべ、追加の条件を付け加える。

### ダイアログ

**リリィ**：
「器が完成したら、仕上げに**『ヴォイド・プチの粘液』**で内側をコーティングすること。
それでようやく、この異次元の『音』を閉じ込めるための膜が完成します。」

*【表情：期待と不安が混じった表情】*

**リリィ**：
「さあ、作業台へ向かいなさい。
あなたがただの暴力装置ではなく、繊細な手先を持つ『創造者』でもあることを、私に証明してみせるのです。」

彼女はわずかに笑う。

**リリィ**：
「……期待はしていませんが、楽しみにはしていますよ？」

*【表情：意地悪な笑み、尻尾がゆっくりと揺れる】*

## 【シーン4：納品と、報酬の授与】

### 演出・背景描写

**[BGM: 穏やかな音楽、成功の音]**

プレイヤーが製作した器を持ってリリィの元へ戻る。

リリィは差し出された器を目の高さに掲げ、偏光する光を透かして微細な傷一つないかを確認する。

*【視覚演出：器が光を透かし、美しく輝く】*

不意に、彼女の尻尾が満足げに揺れた。

### ダイアログ

**リリィ**：
「……あら。意外ですね。」

*【表情：驚きと満足が混じった表情】*

**リリィ**：
「あんな泥にまみれた手で、これほどまでに滑らかな曲線を描き出すなんて。
合格です。これなら、このアリーナに満ちる『死の残響』を存分に吸い取ってくれるでしょう。」

彼女は器を丁寧に棚に置き、台帳を開く。

**リリィ**：
「これはお礼です。」

*【表情：珍しく、柔らかい微笑み】*

**リリィ**：
「観客からの報酬として、**小さなコイン7枚**と**プラチナコイン2枚**を台帳に記録いたしました。
それと、この器を使って精製した素材を、一つ選んでいただけます。」

### プレイヤーの選択肢（報酬素材選択）

**選択肢A**：「魔力の結晶を頼む」
→ リリィ：「『魔力の結晶×1』、記録いたしました。この器で精製した、純粋な魔力の塊ですよ。」

**選択肢B**：「エーテルの欠片が欲しい」
→ リリィ：「『エーテルの欠片×1』、記録いたしました。……慎重にお使いくださいね。」

**選択肢C**：「ガラスの破片を選ぶ」
→ リリィ：「『ガラスの破片×1』ですね。……地味な選択ですが、実用的です。」

---

リリィは台帳を閉じ、あなたを見つめる。

**リリィ**：
「それと……あなたの技術を評価して、称号も記録しておきました。
『繊細な泥犬』。ふふ、あなたは暴力だけではないのですね。」

*【表情：興味深そうな眼差し】*

**リリィ**：
「これからも、何か特別な依頼があれば、あなたにお願いするかもしれません。
……期待していますよ。」

*【表情：微笑み、尻尾がゆっくりと揺れる】*

### プレイヤーの選択肢（PR用）

**選択肢A**：「いつでも頼んでくれ」
→ リリィ：「ふふ、頼もしいこと。では、またお願いいたしますね。」

**選択肢B**：「報酬次第だがな」
→ リリィ：「まあ、現実的ですこと。ええ、もちろんお支払いはしますよ。」

**選択肢C**：「（無言で頷く）」
→ リリィ：「……無口ですが、仕事はきっちりとこなす。良い職人気質ですね。」

## 【システム処理】

**フラグ更新**：
```json
{
  "chitsii.arena.rel.lily": "+5"
}
```

**製作要件**：
- スキル：石細工 または ガラス工芸 Lv10以上推奨
- 素材：黒曜石 または 水晶の塊 × 3
- 追加素材：ヴォイド・プチの粘液 × 5
- 製作目標：『虚空の共鳴瓶』（品質：良質以上）

**報酬**：
- 小さなコイン × 7
- プラチナコイン × 2
- 選択した素材 × 1（魔力の結晶 / エーテルの欠片 / ガラスの破片）

**ステータス変化**：
- 称号獲得：【繊細な泥犬】（製作スキルの成功率+5%）

**コンテンツ解禁**：
- リリィから特殊な依頼が受けられるようになる

---

**【シナリオ意図】**

このシーンは、リリィの「研究者」としての一面を見せる重要なポイントです。
- 戦闘だけではなく、製作技術も評価されることを示す
- リリィとプレイヤーの関係性が「事務員と闘士」から「研究者と協力者」へと深化
- プレイヤーの選択肢を豊富に用意し、ロールプレイの幅を広げる
- 報酬システムを統一形式で提示し、実装をシンプルに
