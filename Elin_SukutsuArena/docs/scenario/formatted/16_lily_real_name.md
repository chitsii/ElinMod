# クエスト：リリィの告白『真名の刻印、永遠の共犯』

**【概要】**
バルガスを救った後、リリィがプレイヤーを私室に招く。そこで彼女は、サキュバスにとって最大の禁忌——自らの「真名」を明かす——という行為に及ぶ。これは単なる感謝ではなく、プレイヤーへの深い愛情と、運命を共にするという覚悟の証である。

**【重要な注意】**
このシーンは、以下の条件を満たした場合のみ発生します：
- scenario/13でリリィとの関係が一定以上（好感度20以上）
- scenario/15でバルガスを救った
- scenario/13で「共鳴瓶の真実」を正直に告白したか、または瓶の交換をしなかった

**【登場人物】**
- **リリィ**：受付のサキュバス。プレイヤーに真名を明かし、運命を共にすることを誓う。

**【シナリオ意図】**

このシーンは、リリィとプレイヤーの関係が最高潮に達する重要なポイントです。
- バルガスを救ったという「行動」が、リリィの心を完全に変えた

- プレイヤーとリリィが「観察者と被験者」から「共犯者」へと関係が変化する
- 真名の契約により、最終決戦でのリリィの支援が大幅に強化される
- プレイヤーに「守られる存在」ではなく「共に戦う伴侶」としてのリリィを提示
- ロマンス要素を強化し、プレイヤーの感情移入を深める
