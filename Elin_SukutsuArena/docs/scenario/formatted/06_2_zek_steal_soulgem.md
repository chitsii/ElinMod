# クエスト：Rank Dへの岐路『英雄の残響、商人の天秤』

**【概要】**
カイン戦の直後、ゼクがプレイヤーの前に現れ、「カインの魂の欠片」を売るよう要求する。これは友情と力のトレードオフであり、プレイヤーの価値観が試される重大な分岐点である。

**【登場人物】**
- **ゼク**：次元の狭間を漂う商人。カインの魂の欠片を狙っている。
- **バルガス**：アリーナマスター。カインの魂の帰還を待っている。
- **リリィ**：受付のサキュバス。プレイヤーの選択を冷静に観察する。

---

## 【シーン1：砂上の囁き】

### 演出・背景描写

**[BGM: 突然の静寂 → 不穏な弦楽器の音]**

闘技場の熱気が冷めやらぬ、薄暗い廊下。

*【視覚演出：廊下の影が不自然に伸び、揺らめく】*

勝利の余韻に浸るあなたの前に、鎖の音と共にゼクが現れる。

*【聴覚演出：鎖の擦れる音、金属的な響き】*

彼は、あなたの掌にある『カインの魂の欠片』を、まるで極上の宝石を見るような飢えた目で見つめていた。

### ダイアログ

**ゼク**：
「……あぁ、なんと美しい。」

*【表情：フードの奥から、恍惚とした表情】*

**ゼク**：
「数千回の敗北と、最期の瞬間の安らぎが凝固した、混じり気なしの『純粋な魂』だ。」

彼は細長い指を伸ばし、魂の欠片を指し示す。

**ゼク**：
「バルガスがそれを望んでいるのは分かっています。……ですが、闘士殿。
あのおっさんにこれを返して、一体何になるのですか？」

*【表情：冷笑、フードの奥で何かを企む目】*

**ゼク**：
「死者に安らぎを、生者に感傷を。……そんなもの、一文の得にもなりゃしない。」

### プレイヤーの選択肢（PR用）

**選択肢A**：「……何が言いたい」
→ ゼク：「おや、警戒心がおありで。では、単刀直入に申し上げましょう。」

**選択肢B**：「友情に価値がないと？」
→ ゼク：「価値がない、とは言いません。ただ、『足りない』と言っているのです。」

**選択肢C**：「（無言で聞く）」
→ ゼク：「……ふふ、沈黙は賢明さの証。では、続けさせていただきましょう。」

## 【シーン2：残酷な二択】

### 演出・背景描写

**[BGM: 不協和音が強まる、心臓の鼓動のような低音]**

ゼクはさらに一歩踏み込み、フードの下から歪んだ笑みを覗かせる。

### ダイアログ

**ゼク**：
「それよりも、私にその欠片を預けてはいただけませんか？
これほどの素材があれば、私はあなたの武具に『神話』の一片を刻むことができる。」

*【表情：誘惑的な笑み、フードの奥で何かを愉しむ目】*

**ゼク**：
「選ぶのはあなただ。
バルガスに返し、友情と名誉という名の、腹の足しにもならない温もりを噛み締めるか。
それとも、私に売り払い、この先の地獄を生き抜くための『絶対的な暴力』を手に入れるか。」

*【表情：冷たい笑み、プレイヤーを見つめる】*

**ゼク**：
「バルガスに返せば、彼は救われるでしょう。ですが、あなたは弱いままだ。
私に売れば、あなたは英雄の力を食らって強くなる。
……どちらがこのアリーナの『正解』か、賢明なあなたならお分かりでしょう？」

### プレイヤーの選択肢（PR用）

**選択肢A**：「代償は何だ？」
→ ゼク：「代償は……あなたの『良心』を、少しばかりいただくだけです。ああ、それと友情も。」

**選択肢B**：「バルガスが知ったら……」
→ ゼク：「ふふ、知られなければいいのですよ。それとも、正直に告白なさいますか？」

**選択肢C**：「（無言で魂の欠片を見つめる）」
→ ゼク：「……悩んでおられる。それが正常です。人間らしいですね。」

「……残念です。ですが、無理強いはいたしません。

**【シナリオ意図】**

このシーンは、プレイヤーの価値観が試される最も重要な分岐点の一つです。
- 友情と力のトレードオフという古典的なジレンマを提示
- プレイヤーの動機（scenario/01で選択）に応じた内省で、選択に深みを与える
- どちらを選んでも一長一短があり、「正解」がないことを示す
- 後のシナリオ（scenario/13）でこの選択の結果が回収され、物語に深みを与える
