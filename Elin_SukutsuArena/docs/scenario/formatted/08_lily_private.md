# クエスト：リリィの招待『紫煙と観察者の深淵』

**【概要】**
戦いの傷を癒し、受付に戻ったプレイヤーをリリィが私室に招待する。そこでは、彼女のサキュバスとしての本性——捕食者の本能と、それを抑えようとする葛藤——が露わになる。

**【登場人物】**
- **リリィ**：受付のサキュバス。私室でプレイヤーの魂に惹かれ、捕食者の本能と葛藤する。

---

## 【シーン1：カウンター越しの視線】

### 演出・背景描写

**[BGM: 静かで妖しい音楽、遠くで鎖の音]**

戦いの傷を癒し、戦果を報告しに受付へ向かったあなた。

リリィはいつものように書類を整理しているが、その動きはどこか緩慢だ。

*【視覚演出：リリィの目が、いつもより妖しく光っている】*

彼女の細長い瞳が、あなたの全身を這うように舐め回す。

### ダイアログ

**リリィ**：
「……おかえりなさい。」

*【表情：柔らかい微笑み、しかし瞳は鋭い】*

**リリィ**：

ふふ、でも……今のあなたからは、とても複雑で、芳醇な香りがします。」

*【表情：うっとりとした表情、目を細める】*

彼女は羽根ペンを置き、あなたに近づく。

**リリィ**：
「絶望と、不屈。そして、ほんの少しの『殺意』が混じり合った、熟成した魂の匂い。」

*【表情：陶酔したような表情、尻尾がゆっくりと揺れる】*

**リリィ**：
「……バルガスの酒臭い怒声が響くこの場所では、その香りを十分に楽しめません。
少し、私の『私室』へいらっしゃいませんか？」

*【表情：誘惑的な笑み、目が妖しく光る】*

**リリィ**：
「仕事の話ではありません。ただ、観察者として……
あなたのその魂を、もう少し近くで愛でたいのです。」

*【表情：柔らかい笑み、しかし瞳には危険な光】*

**選択肢C**：「（無言で頷く）」

## 【シーン2：サキュバスの私室】

### 演出・背景描写

**[BGM: 妖艶で危険な音楽、紫煙のような音]**

ロビーの裏手にある重厚な黒檀の扉。
そこを潜った瞬間、大気の重さが変わる。

*【視覚演出：画面が紫色に染まり、部屋全体が妖しい光に包まれる】*

部屋全体を支配するのは、甘美でありながら肺の奥を痺れさせるような「紫煙」の香。

*【聴覚演出：遠くで何かが脈打つ音、低い唸り声のような音】*

床には柔らかな魔獣の毛皮が敷き詰められ、壁には異次元の珍奇な書物や、何かの魂が閉じ込められたと思わしき脈打つ宝石が飾られている。

*【視覚演出：宝石が鼓動するように明滅する】*

リリィは、毒々しいほど鮮やかなソファに深く腰掛け、事務服の襟元をわずかに緩めた。

### ダイアログ

**リリィ**：
「さあ、遠慮なさらず。」

*【表情：妖艶な微笑み、尻尾がゆっくりと揺れる】*

**リリィ**：
「ここはアリーナの法則からも、神々の視線からも隔絶された場所。」

彼女はソファに深く腰掛け、あなたを見つめる。

**リリィ**：
「……今のあなたの戦い、実に滑稽で、そして美しかったわ。
落下物に翻弄されながらも、その瞳から光が消えなかった。
……あの瞬間、あなたの魂がどれほど甘く弾けたか、自覚はありますか？」

*【表情：陶酔したような表情、目を細める】*

### プレイヤーの選択肢（PR用）

**選択肢A**：「……自覚はない」
→ リリィ：「ふふ、そうでしょうね。当事者は気づかないものです。」

**選択肢B**：「俺の魂がそんなに美味しそうか？」
→ リリィ：「ええ。……とても。食べてしまいたいくらいに。」

**選択肢C**：「（無言で聞く）」
→ リリィ：「……無口ですが、その瞳は饒舌ですね。」

彼女の尻尾が、まるで生き物のようにあなたの足首に絡みついた。

### ダイアログ

**リリィ**：
「……あなたは不思議な人。」

*【表情：興味深そうな眼差し、首を傾げる】*

**リリィ**：
「多くの闘士は、ランクDに上がる頃には心が擦り切れて、ただの戦う機械になる。
けれど、あなたからは『意志』の拍動が聞こえるの。」

*【演出：リリィの瞳が、一瞬だけ妖しく輝く】*

サキュバスの本能が、抑えきれずに漏れ出す。

**リリィ**：
「……ねえ、一つ教えて。あなたは、どこまで登るつもり？
その魂が完全に磨き上げられ、最高級の宝石になった時……
それを最初に手にするのは、私であってもいいかしら？」

*【演出：リリィが至近距離まで顔を寄せる】*

あなたは彼女の吐息に、甘美でありながら危険な「何か」を感じ取る。
——それは、捕食者の本能。

*【表情：瞳が妖しく光り、牙がわずかに見える】*

**リリィ**：
「……ふふ、冗談よ。怖がらないで。」

*【表情：笑顔に戻るが、手が震えている】*

彼女は一歩下がるが、その手は微かに震えている。

彼女の尻尾が、苦しげに床を叩いた。

**【シナリオ意図】**

このシーンは、リリィの「サキュバスとしての本性」と「プレイヤーへの感情」の葛藤を見せる重要なポイントです。
- 捕食者の本能と、それを抑えようとする葛藤で、リリィのキャラクターに深みを与える
- 妖艶で危険な雰囲気を演出し、プレイヤーの緊張感を高める
- リリィとプレイヤーの関係性が「事務員と闘士」から「特別な存在」へと深化
- プレイヤーの選択肢を豊富に用意し、感情移入を促進
