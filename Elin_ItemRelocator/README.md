# Elin Item Relocator (日本語)

**Elin Item Relocator** は、設定したフィルター条件に基づいて、インベントリや周囲のアイテムを自動的にターゲットのコンテナ（収納箱など）に移動させる *Elin* 用のMODです。

## 主な機能

- **コンテキストメニュー統合**: すべてのコンテナ（箱、樽など）のコンテキストメニューに「Relocate Items（アイテムを移動）」というオプションを追加します。
- **動的なフィルタリング**:
  - **カテゴリ**: アイテムカテゴリID（例: `food`, `ore`, `weapon`）でフィルタリング。
  - **レアリティ**: 指定したレアリティ以上のアイテムを対象にします。
- **検索範囲の選択**:
  - **Inventory (インベントリ)**: プレイヤーの所持品のみを対象にします。
  - **Zone (ゾーン)**: 現在のマップ（足元など）にあるアイテムと、プレイヤーの所持品の両方を対象にします。
- **設定の保存**: 各コンテナごとの移動設定は、外部のJSONファイル（`Elin_ItemRelocator_Data.json`）に保存されます。セーブデータ自体を変更しないため安全です。


## 使い方

1. ゲーム内で任意のコンテナを **右クリック** します。
2. メニューから **"Relocate Items"** を選択します。
3. 設定画面が表示されます:
   - **Add Filter**: フィルターを追加します（カテゴリまたはレアリティ）。
   - **Scope**: ボタンをクリックして、検索範囲を「Inventory（インベントリ）」と「Zone（ゾーン）」で切り替えます。
   - **Enable/Disable**: フィルターごとの有効/無効を切り替えられます。
4. 設定を行うと移動処理が実行されます（またはウィンドウを閉じて設定を保存します）。

## 設定ファイル

設定は以下の場所に保存されます：
`%USERPROFILE%\AppData\LocalLow\Lafrontier\Elin\Save\Elin_ItemRelocator_Data.json`

## アンインストール

`Package` フォルダから `Elin_ItemRelocator` フォルダを削除するだけです。セーブデータには影響しません。
